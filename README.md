# JDA-ADMIN-PANEL

## HOW TO RUN PROJECT

Below are the instructions on how to run project locally:

```bash
git clone https://github.com/Sijonra/jda-admin-panel-profile.git
yarn
yarn prepare
yarn run dev
```
